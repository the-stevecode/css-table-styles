Librería de Estilos para Tablas

Introducción

La librería de estilos para tablas "table-styles.css" proporciona una serie de estilos predefinidos para mejorar la apariencia de las tablas HTML en tus proyectos web.

Uso Básico

Para utilizar estos estilos en tus tablas, simplemente enlaza el archivo CSS en la sección <head> de tu página HTML: 

            
            <head>
            <link rel="stylesheet" href="table-styles.css">
            </head>

A continuación, se muestra un ejemplo visual de una tabla estilizada con la librería, con tan solo agregarlo a tu proyecto:

# Nombre de Producto Precio unitario Cantidad
1 Refresco de Limón $12 2
2 Refresco de Toronja $11 3
3 Refresco de Tamarindo $10 2
Totales $33 7

Luego, aplica las clases de estilo a tus tablas y celdas HTML según sea necesario. Por ejemplo:

            
                <table class="table">
                    <thead>
                        <tr>
                            <th> # </th>
                            <th>Nombre de Producto</th>
                            <th>Precio unitario</th>
                            <th>Cantidad</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td> 1 </td>
                            <td>Refresco de Limón</td>
                            <td> $12 </td>
                            <td> 2 </td>
                        </tr>
                        <tr>
                            <td> 2 </td>
                            <td>Refresco de Toronja</td>
                            <td> $11 </td>
                            <td> 3 </td>
                        </tr>
                        <tr>
                            <td> 3 </td>
                            <td>Refresco de Tamarindo</td>
                            <td> $10 </td>
                            <td> 2 </td>
                        </tr>
                    </tbody>
                    <tfoot>
                        <tr>
                            <td colspan="2"> Totales</td>
                            <td> $33 </td>
                            <td> 7 </td>
                        </tr>
                    </tfoot>
                </table>

Personalización

Los estilos pueden ser personalizados según tus necesidades modificando el archivo CSS "table-styles.css". Consulta la documentación incluida en el archivo para más detalles.
